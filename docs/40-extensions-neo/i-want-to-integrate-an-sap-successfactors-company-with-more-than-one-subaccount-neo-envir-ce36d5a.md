# I Want to Integrate an SAP SuccessFactors Company With More Than One Subaccount \(Neo Environment\)

I want to integrate an SAP SuccessFactors company with more than one subaccount in SAP BTP.



<a name="loioce36d5a421af49e9b69e97c29189d6a0__section_xvs_n32_2cc"/>

## Issue/Symptom

You want to integrate one SAP SuccessFactors company with more than one subaccount.



<a name="loioce36d5a421af49e9b69e97c29189d6a0__section_bcc_t32_2cc"/>

## Solution

To integrate one SAP SuccessFactors company with one subaccount, see:

1.  [Create an Integration Token for SAP SuccessFactors](https://help.sap.com/viewer/65de2977205c403bbc107264b8eccf4b/Cloud/en-US/6ef1e3bea61b4ad8a4ebf06637f43c96.html)

2.  [Configure the Extension Integration Between SAP SuccessFactors and SAP BTP](https://help.sap.com/viewer/65de2977205c403bbc107264b8eccf4b/Cloud/en-US/09bb734cf5614896a4cdb66f3e1528ec.html)


If you want to integrate more than one subaccount with the same SAP SuccessFactors company, repeat the same procedure for each subaccount.
